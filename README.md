🎮 Game — The Art of Guessing (and Probably Failing)

Welcome to Game, a deceptively simple yet oddly humbling number guessing experience.
It looks innocent. It is not.

🧠 What is this?

Game is a browser-based number guessing project where the computer picks a random number… and you attempt to outsmart it.

Spoiler: the computer is very confident.

Built primarily with HTML and powered by JavaScript, this project also showcases my early exploration into dynamic theming—because guessing incorrectly should at least look good.

⚙️ How it works
The game secretly generates a random number.
You enter your guess.
The game responds with feedback.
You question your life choices.
Repeat until victory (or emotional growth).
✨ Features
🎲 Random number generation (fair… allegedly)
🎨 Dynamic theme changes based on gameplay
⚡ Instant feedback on guesses
🧩 Simple, clean UI built with HTML
🧠 Logic driven by JavaScript (the real MVP)
🚀 Getting Started

No complicated setup. No unnecessary drama.

git clone https://github.com/your-username/game.git
cd game

Then simply open:

index.html

in your browser and start guessing like a pro (or not).
